# 🏨 BookNest: A Hotel Management System

BookNest is a Database Management System (DBMS) course project developed by a team of three students. The system is designed to simplify hotel operations by managing rooms, reservations, guests, employees, and payments through a structured relational database.

---

## 📌 Project Overview

BookNest aims to provide an efficient hotel management solution by allowing administrators to:

- Manage hotel rooms
- Register and manage guests
- Handle room reservations
- Process payments
- Manage employees
- Generate reports

The project focuses on applying DBMS concepts such as database design, normalization, SQL queries, constraints, relationships, and transaction management.

---

## 🎯 Objectives

- Design a normalized relational database
- Implement CRUD operations using SQL
- Maintain data integrity through constraints
- Demonstrate ER modeling and relational schema design
- Apply DBMS concepts in a real-world scenario
